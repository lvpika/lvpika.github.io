在编写Hexo博客文章的时候插入图片

  1. 如果用图床的链接之类的是不显示的

  2. 我百度到的一个解决办法，就是安装一个hexo插件

  3. 然后修改Hexo的配置文件，将 post_asset_folder 的键值改为 true

  4. 然后在执行 hexo new 创建文章的时候，刚安装的插件会在目录下创建按一个同名的文件夹，你把要放在文章中的图片放在这个文件夹中，然后在文章中 调用图片即可 调用图片的语法如下，注意图片名字要加上后缀名
